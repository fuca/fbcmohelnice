DateTimePicker
==============

DateTimePicker Control for Nette Framework

This add-on creates input box to select date and time.

Requirements
------------

- **PHP** 5.3
- **Nette Framework** 2.0 or later
- **jQuery** 1.9.1
- **jQuery Timepicker Addon** 1.2
- **jQuery UI** 1.10.2
